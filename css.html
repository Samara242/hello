Qu'est-ce que le CSS ?

CSS (Cascading Style Sheets) est le langage qui permet de mettre en forme vos pages web : couleurs, tailles, polices, espacements.

Il existe 3 façons d'intégrer du CSS, mais la meilleure pratique est d'utiliser un fichier .css externe lié à votre HTML, surtout pour les sites multi-pages.

Un fichier CSS externe permet de modifier le style de toutes vos pages depuis un seul endroit. C'est la méthode recommandée.

Code HTML de base

Voici le code HTML minimal pour commencer à pratiquer. Copiez-le dans votre fichier index.html :

index.html 
<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Mon premier CSS</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <h1>Mon Site</h1>
  <p id="intro">Bonjour à tous, j'apprends le CSS !</p>
</body>
</html>

Vous pouvez aussi consulter l'exemple en direct.

Formatage du texte

Formater le texte consiste à modifier son apparence : taille, police, style, alignement.

1. La taille — font-size

On utilise la propriété font-size. La valeur peut être en px (pixels), rem, ou %.

style.css 
p {
  font-size: 20px;
}
Rendu dans le navigateur

2. La police — font-family

On utilise font-family. Il est recommandé d'indiquer plusieurs polices pour que le navigateur en choisisse une disponible sur la machine.

style.css 
h1 {
  /* Mettre 3 polices de préférence */
  font-family: 'Times New Roman', Times, serif;
}

3. Italique, Gras, Souligné

Italique — font-style 
body {
  font-style: italic;
}
Rendu
Gras — font-weight 
p {
  font-weight: bold;
}
Souligné / Barré / Surligné — text-decoration 
h1 {
  text-decoration: underline;   /* souligné */
}
p {
  text-decoration: line-through; /* barré */
}
span {
  text-decoration: overline;    /* ligne au-dessus */
}
Rendu

Alignement du texte

La propriété text-align permet de positionner horizontalement votre texte.

style.css 
h1 { text-align: center; }   /* centré */
h2 { text-align: left; }    /* à gauche (défaut) */
h3 { text-align: right; }   /* à droite */
p  { text-align: justify; } /* justifié */
Rendu

Couleur & Fond

Utilisez color pour la couleur du texte, et background-color ou background-image pour le fond.

style.css 
p {
  color: purple;       /* nom de couleur */
}

h1 {
  color: #FFC305;      /* notation hexadécimale */
}

p {
  background-color: blue; /* fond coloré */
}

body {
  background-image: url("fond.png");
  background-repeat: no-repeat; /* pas de répétition */
  background-size: cover;       /* couvre tout l'écran */
}
