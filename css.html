Introduction au CSS

CSS est le langage qui nous permet de mettre en forme nos pages web.Dans ce module nous verrons ensemble comment utiliser le CSS dans nos pages.


Par quoi commencer ?

il faut savoir qu'il existe trois manières différrentes d'utiliser CSS: dans le head, le body en l'inserant directement dans les balises html.
Mais, ces pratiques peuvent être genantes lorsqu'on a un site qui contient plusieurs pages web; alors nous utiliserons un fichier .css externe qui reste la meilleur façon d'utiliser CSS?

Nos pages web sont structurés grâce au langage HTML, pour les rendre plus beau à l'oeil nous utilisons des propriétés du CSS.


le formatage du texte: formater le texte consiste à modifier son apparence.

Voici le code html que nous allons utiliser pour ce cours,faites comme nous

 
                               <!DOCTYPE html>
                               <html lang="fr">
                               <head>
                               <meta charset="UTF-8">
                               <meta name="viewport" content="width=device-width, initial-scale=1.0>
                               <title>Mon premier CS<title>
                               <link rel="stylesheet" href="style.css">
                               <head>
                               <body>
                                    <h1>Mon Site<h1> 
                                    <p id="titre" >bonjour à tous, j'apprends le css.<p>
                               <body>
                               <html>

  1. La Taille : on utilise la propriété font-size: sa valeur en px ou % 

    2. 
                             p {
                               font-size: 20px;
                              }

  2. la police: on utilise la propriété font-familly, on met le nom de la police que l'on souhaite, de préfférence en mettre trois ou quatre le navigateur affichera la police qu'on a dans nos fichiers.

    3. 
                                 h1 {
                                     font-familly: Times NewsRoman, Times, serif;/*mettre trois police de préférence
     pour permettre au navigateur de sélecttionner celle dont le fichier est présent dans notre machine*/
                                    }

  3. Italique, Gras, Souligné

    4. 
                                  body { 
                                        font-style: italic;
                                       }
                                 


    
                                      p { 
                                          font-weight: bold;/*en gras*/
                                        }
                                  
    
                                      h1 { 
                                           text-decoration: underline;/*souligné*/
                                         }
                                  
    
                                        p { 
                                            text-decoration: Line-through;/*barré*/
                                         }
                                  
    
                                        p { 
                                           text-decoration: overline;/*souligné le haut*/
                                        }

  4. Alignement du texte

    5. 
                                       h1 { 
                                            text-align: center;
                                         }
                                   
    
                                        body { 
                                        text-align: right;
                                        }
                                    
    
                                        h1 { 
                                        text-align: left;
                                        }
                                    
    
                                        h1 { 
                                        text-align: justify;
                                        }


  5. La couleur : on utilise la propriété color, pour le texte uniquement et background-color, pour le fond et aussi background-image, pour un fond en image

    6. 
                                            p { 
                                            color:purple;
                                            }
                                            h1{
                                                color: #FFC305;/*la couleur peut s'écrire en hexadécimal*/
                                            }
                                            p {
                                                background: blue;
                                            }
                                            body{
                                                background-image: url("exemples/n30.png");
                                                background-repeat: no-repeat;/*évite que l'image se répète plusieurs fois*/
                                            }
